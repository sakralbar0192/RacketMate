# racketmate

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.2.15. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.

## Слияние с VBallAgregator

Логика RacketMate переносится в монорепозиторий [VBallAgregator](../VBallAgregator) (`packages/bot-racket`, общий домен в `packages/core`). Этот репозиторий оставлен как исторический прототип; новая разработка — в основном проекте.
